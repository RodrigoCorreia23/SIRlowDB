Documentação do Projeto

Este projeto implementa uma API RESTful utilizando Node.js, Express e a biblioteca lowDB para simular a gestão de um recurso de "Estudantes". A API permite realizar operações CRUD (Criar, Ler, Atualizar e eliminar) numa base de dados, que é armazenada em um arquivo JSON utilizando lowDB.

    Funcionalidades principais:

  • Gestão de Estudantes: A API permite inserir, visualizar, editar e eliminar estudantes. Cada estudante possui um identificador único (ID), nome e curso e ano de matrícula.
  • Persistência de Dados: A API utiliza lowDB para garantir que os dados sejam armazenados Num arquivo JSON, mantendo a persistência mesmo após reiniciar o servidor.

    • Funcionalidades principais:

  • GET /students Retorna uma lista com todos os estudantes inseridos.
  • GET /students/{id} Retorna os detalhes de um estudante específico, identificado pelo ID.
  • POST /students Cria um novo estudante. Requer que os campos nome, curso e ano sejam fornecidos.
  • PUT /students/{id} Atualiza os dados de um estudante específico. Requer que o ID do estudante e os campos a serem atualizados sejam fornecidos.
  • DELETE /students/{id} Remove um estudante específico com o ID fornecido.
